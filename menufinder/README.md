# 🔍 메뉴파인더 (MenuFinder) - 대학생 인증 맛집 검색 플랫폼

대학생들의 얇은 지갑 사정과 라이프스타일을 반영한 **원하는 메뉴 기반 식당 추천 및 영수증 인증 평점 서비스**입니다.  
별도의 백엔드 서버 없이 웹 브라우저 단독으로 실행 및 데이터 보관(`localStorage`)이 가능한 정적 웹앱(Single Page Application) 형태로 제작되었습니다.

---

## ✨ 핵심 제공 기능

1. **학우 인증 및 회원 가입**
   - 대학교 메일 인증 컨셉의 회원가입/로그인.
   - 브라우저를 종료해도 자동으로 로그인 상태가 유지됩니다.
2. **원하는 실제 메뉴 기준의 추천 검색**
   - 찌개, 돈카츠, 삼겹살 등 식당의 **실제 메뉴판 목록** 데이터를 기반으로 실시간 부분 매칭.
3. **대학생 맞춤형 3대 필터**
   - **💸 가성비 필터**: `8,000원 이하` / `10,000원 이하` 선택 매칭 메뉴 필터링.
   - **🍱 혼밥 가능 필터**: 1인석 완비 및 1인 메뉴 주문이 가능한 매장 필터링.
   - **👥 단체석 완비 필터**: 과/동아리 뒤풀이 및 회식을 위한 20인 이상 수용 & 예약 매장 필터링.
4. **영수증 인증 리뷰 & 신뢰 평점**
   - 영수증 스캔 및 국세청 전송 내역 조회 시뮬레이터를 통해 **신뢰도 100% 영수증 인증 리뷰** 작성.
   - 인증 리뷰 점수가 반영된 별점(⭐)이 실시간으로 식당 카드 및 목록에 반영됩니다.

---

## 📂 파일 구조 및 구성

- **`index.html`**: 싱글 페이지 웹앱(SPA) 가상 라우팅 화면 구조.
- **`style.css`**: HSL 기반 오렌지 피치 디자인 시스템 및 인터랙션 애니메이션.
- **`data.js`**: 안암(고려대) 맛집 8곳의 상세 메뉴판, 가격대, 위치, 단체수용력, 초기 영수증 리뷰 데이터.
- **`app.js`**: 계정 관리(로컬스토리지), 필터/정렬 연산, 영수증 스캔 시뮬레이터 로직.

## 🚀 로컬 실행 방법

1. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 더블클릭하여 바로 엽니다.
2. 혹은 VS Code의 **Live Server** 익스텐션을 활성화하여 로컬 서버 모드(`http://127.0.0.1:5500`)로 실행합니다.
3. 최초 로그인 시에는 회원가입 탭을 눌러 계정을 생성하시거나, 테스트용 계정을 사용해 보세요:

   - **비밀번호:** `123`

## 🌐 GitHub Pages 배포 가이드 (배포하기)

본 프로그램은 순수 정적 웹 리소스(HTML, CSS, JS)로만 이루어져 있으므로 **GitHub Pages를 통해 무료로 호스팅**할 수 있습니다.

### 1단계: Git 저장소 초기화 및 커밋
로컬 터미널을 열고 `menufinder` 디렉토리로 이동한 뒤 아래 커맨드를 입력합니다:
```bash
# Git 초기화
git init

# 모든 파일 추가
git add .

# 첫 커밋 작성
git commit -m "Initialize MenuFinder web application"
```

### 2단계: GitHub 원격 저장소 연결
GitHub 웹사이트에서 새로운 리포지토리(`menufinder`)를 생성한 뒤, 주소를 복사하여 연결합니다:
```bash
# 원격 브랜치를 main으로 설정
git branch -M main

# GitHub 원격 저장소 연결 (YOUR_USERNAME과 REPO_NAME을 실제 본인 값으로 대체)
git remote add origin https://github.com/YOUR_USERNAME/menufinder.git

# 코드 업로드
git push -u origin main
```

### 3단계: GitHub Pages 활성화
1. GitHub 웹 브라우저의 해당 리포지토리 페이지로 이동합니다.
2. 상단의 **Settings** 탭을 클릭합니다.
3. 좌측 사이드바에서 **Pages** 메뉴를 클릭합니다.
4. **Build and deployment** 섹션의 Source에서 **Deploy from a branch**를 선택합니다.
5. Branch 설정에서 `main` 브랜치와 `/ (root)` 폴더를 선택한 뒤 **Save** 버튼을 누릅니다.
6. 1~2분 뒤 페이지 상단에 배포 완료 링크가 생성됩니다!
   - 예시 주소: `https://YOUR_USERNAME.github.io/menufinder/`
